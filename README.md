Project Management Tool

A full-stack Project Management Tool built using React, Node.js, Express, and SQLite that helps users manage projects, tasks, and workflows efficiently. The application provides a simple and intuitive interface for organizing work, tracking progress, and collaborating within a team.

This project demonstrates modern full-stack web development concepts, including REST APIs, authentication, database integration, and state management.

🚀 Features
🔐 User Authentication (Register & Login)
📋 Create and Manage Projects
✅ Task Management System
🗂 Organize tasks by project
🌐 RESTful API integration
💾 SQLite Database Storage
⚡ Fast React Frontend
🔒 Secure backend using Express middleware
📱 Responsive UI
